# The L-System Arboretum

A web-based incremental idle game where you cultivate a procedurally generated fractal tree using L-System genetics.

## Documentation

- [**Gameplay Guide**](./docs/GAMEPLAY_GUIDE.md): Learn how to play, understand resources, and master the seasons.
- [**Game Design**](./idea.md): The original concept and technical architecture.

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run the development server:
    ```bash
    npm run dev
    ```
3.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
